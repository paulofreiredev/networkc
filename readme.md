# NetworkC

Estou estudando conceitos de rede e sistemas operacionais utilizando C apenas por interesse em saber como as coisas funcionam
Esse projeto contém exemplos de código apenas para referência.
O que estudei aqui pode ser encontrada no livro de redes do Beej que ele disponibiliza gratuitamente no [site](https://beej.us/guide/bgnet/)

Para executar você precisa do compilador gcc que vem no pacote build-essentials. Instale e é possivel executar os exemplos 
usando gcc arquivo.c -o bin/executavel

O que cada um faz está documentado como comentário no proprio arquivo, mas sugiro que não use esse projeto pra estudar e estude pelo livro.
